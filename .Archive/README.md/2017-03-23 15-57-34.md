##询问和查找重要结论的原始出处
在古典老师的文章中看到一个重要的观点或规律，我不仅仅要记住结论，还要更进一步做功课——寻找这个结论或规律的原始出处。
当你知道这个结论的原始出处，你对这个结论的适用范围和可能的局限性就会有更清晰的认识。
